📖 Google Apps Script Setup — True Live Sync
✓ Good news: Google Apps Script runs on Google's servers (free) and can access your Gmail, so no backend hosting is needed. Setup takes ~5 minutes.
Quick Setup (5 steps)
1
Download the Apps Script file
Click the button below to download gmail-sync-script.gs — a complete, ready-to-paste script.
2
Open Google Apps Script
Go to script.google.com → click "New project"
3
Paste the script & save
Delete the default code → paste the downloaded file → rename the project to "Job Search Gmail Sync" → press Ctrl/⌘+S to save.
4
Run initial setup
From the function dropdown (top toolbar), select initialSetup → click ▶ Run → grant Gmail + Drive permissions when asked.
Then select syncGmailToSheet → click ▶ Run (this does your first import).
Then select installHourlyTrigger → click ▶ Run (this schedules auto-sync every hour).
5
Publish the Sheet as CSV
Open your Google Drive → find the new sheet "Job Search Leads" → File → Share → Publish to web → choose the sheet + Comma-separated values (.csv) → click Publish → copy the URL.
Then go to the Live CSV Sync tab here and paste that URL.
💡 What the script does:
• Scans your Gmail every hour for job-related keywords (configurable)
• Extracts company, HR name, email, phone, subject, status (Offer/Rejected/Round 1 etc.)
• Writes each unique email as a row in your sheet (no duplicates — uses Gmail message ID)
• Applies a "JobBI-Processed" label to prevent re-scanning
• Color-codes status cells in the sheet (green = Offer, red = Rejected, etc.)
🔒 Privacy note: The script runs in your Google account on Google's servers. No data leaves Google's ecosystem. The CSV URL works because Google's "Publish to web" creates a public-readable endpoint — only use it if you're comfortable with that Sheet being public. For a private workflow, use the .mbox import instead.
